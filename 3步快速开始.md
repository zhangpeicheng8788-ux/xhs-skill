# 小红书批量发布工具 - 3步快速开始

## 🚨 遇到登录错误？

看到这个错误信息：
```
{'code': -1, 'success': False}
{'code': -100, 'msg': '无登录信息', 'success': False}
```

**别担心！按照下面3步操作即可：**

---

## ⚡ 第1步：重新登录（2分钟）

**双击运行：**
```
fix_cookie.bat
```

**会发生什么：**
1. 浏览器自动打开小红书登录页面
2. 用小红书APP扫描二维码
3. Cookie自动保存

---

## ⚡ 第2步：启动工具（10秒）

**双击运行：**
```
start_publish_fixed.bat
```

---

## ⚡ 第3步：发布笔记（1分钟）

**在打开的窗口中：**
1. 点击"浏览"选择文件夹（比如 `D:\jieyue_work`）
2. 点击"检测笔记"扫描所有笔记
3. 点击"开始发布"开始批量发布

**就这么简单！** ✨

---

## 📁 支持的文件夹结构

工具会**自动检测所有子文件夹**中的笔记！

### 示例1：平铺式
```
D:\jieyue_work\
├── note_1\
│   ├── cover.png      ← 必需
│   ├── card_1.png
│   └── card_2.png
├── note_2\
└── note_3\
```

### 示例2：分组式
```
D:\jieyue_work\
├── drivingschool_notes\
│   ├── note_01\
│   └── note_02\
└── spring_notes\
    └── note_01\
```

### 示例3：多层嵌套
```
D:\jieyue_work\
└── batch_1\
    └── group_a\
        └── note_01\
            ├── cover.png
            └── card_1.png
```

**只要文件夹里有 `cover.png` 就会被检测到！**

- ✅ **智能去重**：自动跳过已发布的笔记

## 🔍 常见问题

### Q1：提示"Cookie已失效"？
**答**：Cookie过期了，运行 `fix_cookie.bat` 重新登录。

### Q2：检测不到笔记？
**答**：确保笔记文件夹包含 `cover.png`，工具会扫描所有子文件夹。

### Q3：如何查看已发布的笔记？
**答**：点击窗口中的"发布记录"按钮。

### Q4：如何重新发布某个笔记？
**答**：删除笔记文件夹中的 `.published` 文件。

### Q5：会重复发布吗？
**答**：不会，工具会自动跳过已发布的笔记。
