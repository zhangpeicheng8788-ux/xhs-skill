## 🚀 使用方法

### 克隆仓库
```bash
git clone https://github.com/zhangpeicheng8788-ux/xhs-skill.git
cd xhs-skill
```

### 快速开始
```bash
# 1. 重新登录
fix_cookie.bat

# 2. 启动工具
start_publish_fixed.bat

# 3. 在GUI中操作
#    - 点击"浏览"选择路径（如 D:\jieyue_work）
#    - 点击"检测笔记"扫描所有笔记
#    - 点击"开始发布"启动批量发布
```

### 查看赞赏二维码
1. 启动工具后，在界面顶部可以看到作者信息
2. 点击"💰 赞赏作者"按钮
3. 弹出窗口显示收款二维码
4. 扫码赞赏
