## 📝 使用说明

### 查看作者信息

## ⚠️ 注意事项

### 1. 依赖要求
需要安装 Pillow 库来显示二维码图片：
```bash
pip install pillow
```

### 2. 二维码文件
二维码文件位置：
```
D:\20260127XHS\Auto-Redbook-Skills-main\sang.jpg
```

如果文件不存在，赞赏窗口会显示错误提示。

### 3. 自动安装
使用 `start_publish_fixed.bat` 启动时，会自动检查并安装 Pillow。
